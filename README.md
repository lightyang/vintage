Vintage
==
A quick and dirty Ethereum implementation in Scala.

Done:
- RLP
- Trie
- VM (missing precompiled contract support)

TODO:
- Transaction
- Blockchain

This currently passes all VM Tests in [Ethereum Tests](http://ethereum-tests.readthedocs.io/en/latest/index.html).
